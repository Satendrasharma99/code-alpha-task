# Simple E-commerce Store

This project is a full-stack E-commerce store developed as part of Task 1 for the CodeAlpha Internship.

## Features
- **Product Listing:** View all available products and their details.
- **Shopping Cart:** Add, remove, and manage items in the shopping cart.
- **User Authentication:** Secure login and registration functionality for users.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript / TypeScript
- **Backend & Database:** Replit Agent Environment
